# 納品品質化 ToDo

このリストは、現在の `Real Estate Field Copy Helper` を高機能プロトタイプから納品品質に近づけるための作業メモです。

## 前提

- 対象は仲介向け賃貸サイトを中心にする。
- 一般向け賃貸サイトは検証用として扱う。
- AI補助は一発自動抽出ではなく、ユーザーがコピペした情報をもとにAIと一緒に表を完成させる補助機能として設計する。
- テーブル型、カード型、div型、混在型のすべてを想定するが、テーブル型へ過度に最適化しない。
